# Prerequisites
- Node >= 10.0
- NPM >= 6.0

# How can I use it
- npm install
- npm start
- evaluating hands in a file: npm start -- --readFromFile [filename]
    - file must follow the following format:
    ```
    JJJQQ KKKTT
    55566 44447
    ........
    ```
    - example input file: input.txt, you can try it by `npm run start:file`

# Tests
